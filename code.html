Mickaël Brasebin

Chercheur en sciences de l'information géographique

Les codes nécsesaires à mes recherches sont tous déposés en Open-Source. Cette démarche s'inscrit tout d'abord dans un souci de reproductibilité, mais également de partage avec les équipes de recherche et les acteurs publiques avec lesquels je co-construis les méthodes proposées.

SIG3D

Nom Contenu Adresse
GeOxygene Plateforme SIG3D pour la manipulation et la visualisation de données géographiques SIG3D. https://github.com/IGNF/geoxygene/
GeOxygene-SIG3D-appli Applications développées à partir des fonctionnalités de GeOxygene3D (calcul d'ouverture de ciel, de profils de rue, géénration d'enveloppes constructibles...). https://github.com/IGNF/geoxygene-sig3d-appli

SimPLU3D : génération de formes bâties

Nom Contenu Adresse
SimPLU3D-rules Module conenant le modèle de données géographiques, le modèle de règles et l'importeur de données géographiques. https://github.com/IGNF/simplu3D-rules
SimPLU3D Module permettant de générer des formes bâties à partir de règles issues de Plans Locaux d'Urbanisme https://github.com/IGNF/simplu3D
SimPLU3D-OCL Module permettant de gérer les règles d'urbanisme avec le langage OCL https://github.com/IGNF/simplu3D-OCL
SimPLU3D-OpenMole Module contenant le couplage avec la plaeforme d'exploration de modèles OpenMole et les méthodes pour exploiter les résultats https://github.com/IGNF/simplu3D-OpenMole
